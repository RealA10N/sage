Expand description

A core instruction of the assembly language. These are instructions guaranteed to be implemented for every target possible.

Variants

Comment(String)

Many(Vec<CoreOp>)

Set(Location, i64)

Set the value of a register, or any location in memory, to a given constant.

SetLabel(Location, String)

Set the value of a register, or any location in memory, to the value of a label’s ID.

GetAddress

Fields

addr: Location
dst: Location

Get the address of a location, and store it in a destination

Call(Location)

Get a value in memory and call it as a label ID.

CallLabel(String)

Call a function with a given label.

Return

Return from the current function.

Fn(String)

Declare a new label.

While(Location)

Begin a “while the value is not zero” loop over a given register or location in memory.

If(Location)

Begin an “if the value is not zero” statement over a given register or location in memory.

Else

Add an “else” clause to an “if the value is not zero” statement.

End

Terminate a function declaration, a while loop, an if statement, or an else clause.

Move

Fields

src: Location
dst: Location

Copy a value from a source location to a destination location.

dst = src

Copy

Fields

src: Location
dst: Location
size: usize

Copy a number of cells from a source referenced location to a destination referenced location.

This will dereference src, copy its contents, and store them at the location pointed to by dst.

*dst = *src

Swap(Location, Location)

Swap the values of two locations.

Next(Location, Option<i64>)

Make this pointer point to the next cell (or the nth next cell).

Prev(Location, Option<i64>)

Make this pointer point to the previous cell (or the nth previous cell).

Index

Fields

src: Location
offset: Location
dst: Location

Get the address of a location indexed by an offset stored at another location. Store the result in the destination register.

This is essentially the runtime equivalent of dst = addr.offset(offset)

Inc(Location)

Increment the integer value of a location.

Dec(Location)

Store the comparison of “a” and “b” in a destination register. If “a” is less than “b”, store -1. If “a” is greater than “b”, store 1. If “a” is equal to “b”, store 0.
